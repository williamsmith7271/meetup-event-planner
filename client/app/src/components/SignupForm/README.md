## SignupForm Component
A signup form component.

### Example

```js
<SignupForm
  {...fields}
  onSubmit={this.handleSubmit}
/>
```

### Props

| Prop          | Type     | Default     | Possible Values
| ------------- | -------- | ----------- | ---------------------------------------------
| **onSubmit**    | Func   |             | A callback function called on submission
